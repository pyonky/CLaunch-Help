Keyboard (Options dialog)
Keyboard Operations
Allows you to change the key assignments for basic operation shortcut keys.
Select a classification from the Categories list, choose the function you want to modify from the Commands list, and then enter the desired key in Assigned Key.

Start searching by typing a character key in the main window
Even when the search window is not displayed, typing a character key while the CLaunch window is active will display the search window and start incremental search immediately. However, if a single character key is assigned to a basic operation or an item-specific shortcut key, that assignment takes priority. Also, repeated input of the same character key cannot be used as a shortcut key.
When this option is enabled, the search window closes automatically when all search text is deleted.

Select the first button in the next row when looping
When the rightmost button in a row is selected and you press the right arrow key, the selection normally loops to the leftmost button in the same row. If this option is enabled, the leftmost button in the next row will be selected instead.

Opposite-side button selection without scrolling
If registered buttons exceed the visible window area, pressing the down arrow key while the bottom button is selected normally scrolls the window to reveal hidden items. If this option is enabled, the selection loops to the first row without scrolling. (Scrolling with Ctrl + arrow keys is still available.)

Assign shortcut key automatically when item is registered
When registering a file with an alphanumeric name, the first character of the file name is automatically assigned as the item shortcut key.

Wait for timeout of consecutive key presses
Allows different items to be assigned to the same key combination based on the number of consecutive presses.
For example, if pressing Ctrl twice launches item A and pressing Ctrl three times launches item B:
If this option is disabled, pressing Ctrl three times will launch both item A and item B.
If this option is enabled, pressing Ctrl three times launches only item B. Pressing Ctrl twice launches item A after the specified timeout has elapsed.

Hotkey for displaying CLaunch
Specifies the keyboard operation used to display the CLaunch window.
After entering the desired key, use the spin button on the right to specify how many consecutive presses are required for the key to be recognized as a hotkey.

Initial Page
Click Change to open the Initial Page dialog, then select the page you want to display first. If you select Last Displayed Page, the page that was visible before CLaunch was hidden will be displayed.

Hide CLaunch when the cursor leaves the window
After CLaunch is shown by a click function, it will automatically be hidden when the cursor leaves the window.
