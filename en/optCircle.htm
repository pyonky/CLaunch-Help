Circles (Options dialog)
Circle Functions
Allows you to show CLaunch or launch registered items when you draw a circle with the mouse.
You can assign functions separately to clockwise and counterclockwise circle gestures.

When you enable Assign, the Initial Page dialog appears. Select the page you want to display first.
If you select Last Displayed Page, the page that was visible before CLaunch was hidden will be displayed.
To change the initial page, uncheck the option once and then check it again to reopen the Initial Page dialog.
If the selected direction is already assigned to mouse direct launch, the current assignment will be cleared and reassigned to showing CLaunch.
Mouse direct launch for individual items can be configured from Mouse Launch in the Item Properties dialog.
Number of revolutions
Specifies how many times you must draw a circle. Setting a lower value increases the chance of unintended activation.
Maximum radius
Specifies the maximum radius of the circle. Circles larger than the specified value are ignored.
Setting a larger value increases the chance of unintended activation.
Minimum radius
Specifies the minimum radius of the circle. Circles smaller than the specified value are ignored.
Setting a smaller value increases the chance of unintended activation.

Common Options
Hide CLaunch when the cursor leaves the window
After CLaunch is shown by a circle function, it will automatically be hidden when the cursor leaves the window.

Disable the circle functions while the left button is pressed
Disables circle functions while the left mouse button is pressed.

Disable the circle functions while the right button is pressed
Disables circle functions while the right mouse button is pressed.

Disable the circle functions while the middle button is pressed
Disables circle functions while the middle mouse button is pressed.

* The options that disable circle functions while a mouse button is pressed may not work correctly in environments where a resident application (such as mouse gesture software) hooks mouse button press events.
