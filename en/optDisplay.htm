Display (Options dialog)
Display positions
Select the initial display position of CLaunch for each category of display operation.
Select an operation category from the Operation to Show CLaunch drop-down list, then select the initial position for that operation from the Position list.
  • Around the Cursor - Displays CLaunch centered around the cursor.
  • Bottom Right of the Cursor - Displays CLaunch at the bottom-right of the cursor.
  • Previous Position - Displays CLaunch at the position where it was shown before being hidden.
  • Center of the Screen - Displays CLaunch at the center of the primary display.
  • Specified Coordinates - Displays CLaunch at the coordinates specified below.

X coordinate, Y coordinate
If Specified Coordinates is selected, specify the display position of CLaunch using the display’s X and Y coordinates.
These coordinates are applied to all show operations. You cannot specify different coordinates for each operation category.

Always on top
Displays the CLaunch window above all other windows.

Only while pinned
When CLaunch is pinned, it stays on top. When it is unpinned, the topmost state is removed.

Standby Position
Settings for keeping CLaunch always visible on the screen.
Check Always show CLaunch, then select the desired location from the Position list.
When you perform an operation assigned to show CLaunch (such as a click function or hotkey), CLaunch moves to the position associated with that operation and becomes active.
When the cursor leaves the window or CLaunch becomes inactive, it returns to the standby position.
If you click the Close button or press the Esc key, CLaunch is hidden.

Tooltips
If Show tooltips is checked, tooltips are displayed when you keep the cursor over a registered item.

Include tabs
Displays page names in a tooltip when you keep the cursor over a page tab.

Balloon-shaped
Displays tooltips in a balloon style.

Delay
Specifies how long the cursor must remain stationary before a tooltip is displayed.

Max Width
Limits the width of tooltips to the specified value and wraps text that exceeds the width.
However, if the tooltip contains a word longer than the specified width, the tooltip width will expand to fit that word.
Specify 0 for unlimited width.

Tray icon
If you do not want to display an icon in the task tray, uncheck Show tray icon.

Use alternative icons
Lets you change the icon displayed in the task tray.
Specify a DLL or ICL file that contains at least five 16×16 pixel icons.
The correspondence between icon indexes in the file and displayed icons is as follows.
The number of animation frames depends on how many icons the file contains.

0: Normal state
1: Restricted Mode / Hooks disabled
2: Screen saver suspended
3 ~: First and subsequent frames of animation
