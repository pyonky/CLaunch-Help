Showing and Hiding CLaunch
Show CLaunch
With the default settings, CLaunch can be displayed by double-clicking on the desktop or by moving the cursor to the bottom-right corner of the screen while holding down a mouse button.
In addition, you can choose any of the following actions as a display method in the options.
Single, double, or triple clicks, wheel rotation, or press-and-hold actions using individual mouse buttons or their combinations
Click Functions
Moving the cursor to any of the four screen edges or corners
Edge Functions
Drawing a circular gesture with the mouse
Circle Functions
Entering a predefined hotkey
Hotkeys
Multiple display actions can be assigned, and the initial page to be shown can be specified individually for each action.
(For example, left-click shows the first page, while wheel-click shows the second page.)
The effective range of click functions can be limited to the Title Bar, Desktop, Folder background, or the four screen edges and corners.
You can also set the range to Anywhere to allow the launcher to be displayed regardless of cursor position.
Hide CLaunch
CLaunch is hidden from the screen by the following actions.
Moving the cursor outside the CLaunch window
(Only when this option is enabled)
Clicking on the desktop or another application
Pressing the Escape key
