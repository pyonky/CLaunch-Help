Startup Options
Command Line Parameters
You can specify the following command line parameters when starting CLaunch.

/n:Starts CLaunch in non-resident mode.
CLaunch is displayed at startup and exits when it is hidden.
By default, it is initially displayed at the position where CLaunch was last closed, but you can specify the display position by combining /m or /s.
/nh:Starts CLaunch in the Unhook state.
/l:Displays CLaunch in the pinned state.
By default, it is initially displayed at the position where CLaunch was last closed, but you can specify the display position by combining /m or /s.
/m:Displays CLaunch at the current cursor position.
/s:Displays CLaunch at the specified coordinates.
Format : /s[X coordinate]:[Y coordinate]
Example : /s200:100
/p:Specifies the initial display page.
Format : /p[page number]
Example : /p2
/b:Specifies the initial number of buttons.
Format : /b[horizontal]:[vertical]
Example : /b5:3
/d:Delays the startup of CLaunch by the specified time (in milliseconds).
Format : /d[wait time]
Example : /d3000 (3-second delay)
/a:Enables the auto-hide function.
After CLaunch is displayed by /n, /l, /m or /s, it will automatically be hidden when the cursor moves outside the window.
/r:Discards the cache data and re-acquires the icon images.
Launching Multiple Instances of CLaunch
If CLaunch is already resident and CLaunch.exe is launched again, the CLaunch window will be displayed. (It will be forcibly displayed even in Restricted Mode or in the Unhook state.)
In this case, the command line parameters /l, /m, /s, /p, /a can be specified for the subsequently launched CLaunch.exe.
