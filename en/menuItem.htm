Item Menu
Displayed when you right-click an item button or an item inside a submenu.
Menu items specific to program and document items
Run as Administrator
Launches the target program with administrator privileges.
This menu item appears when CLaunch is running with standard user privileges, or when Run as Standard User is selected for the Privilege Level in the Item Properties dialog.
(Not displayed when UAC is disabled or on Windows XP and earlier.)

Run as Standard User
Launches the target program with standard user privileges.
This menu item appears when CLaunch is running with administrator privileges, or when Run as Administrator is selected for the Privilege Level in the Item Properties dialog.
(Not displayed when UAC is disabled or on Windows XP and earlier.)

Clipboard Launch
Launches the target program using the file path currently copied to the clipboard as a parameter.
For document items, the associated application is used.

Menu items specific to folder items
Open
Opens the folder in the system default folder window.

Open in Explorer
Opens the folder in Explorer with the navigation pane displayed.

Open as Menu
Displays the folder contents as a menu.

Menu items specific to submenu buttons
Register Item
Registers a new item in the submenu.
The Register Item dialog appears.

Create Child Menu
Adds a child submenu.

Sort Items
Sorts items in the submenu by name.

Launch all items in this menu
Launches all registered items in the submenu.
Additional Functions special items and items in child submenus are excluded.

Open Parent Folder
Opens the folder containing the target file.
Open Parent Folder as Menu
Displays the parent folder as a menu.
Hidden files are also shown.
Show Shell Menu
Displays the shell context menu for the target file.

Lock This Button
Locks the button.
Insert Empty Button
Shifts buttons after the target position and inserts an empty button.
If Item Fill Direction in the Design 2 tab of the Options dialog is set to Horizontal, buttons shift right; if Vertical, they shift downward.
Delete
Deletes the item registration.
Row/Column Operations
Allows inserting an empty row or column.
Insert Empty Row shifts rows downward from the selected button.
Insert Empty Column shifts columns rightward.

Exclude from History (history menu only)
Excludes the specified item from usage history.
Excluded items are registered in the Items to exclude from history list in the History Management dialog.

Properties
Displays the Item Properties dialog to modify item settings.
