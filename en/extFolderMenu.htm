Displaying Folder Contents as a Menu
Displaying the contents of a folder as a menu
When a folder is registered to a button, its contents can be displayed by expanding them as a menu.
To display a folder as a menu, set Methods to open folders in the item properties to Open as Menu, or right-click the button with the registered folder and select Open as Menu from the Item Menu.
Files displayed in the menu are opened with a single click, while folders are opened with a double-click. (When multiple items are selected, files are also opened with a double-click.)
Right-clicking a menu item displays the Explorer context menu.
Right-clicking an empty area of the menu displays the General Menu.
Other operations are the same as for submenus.
Submenu Operations
Dragging and dropping menu items
Menu items can be dragged and dropped onto other menu items, item buttons on a page, or other applications. Files dragged from Explorer can also be dropped onto menu items.
Dragging a menu item onto a program button on a page launches the program with the dropped file as a parameter.
Dragging a menu item onto a folder button on a page, a folder within the menu, or Explorer allows files to be copied or moved.
When dragging a file and moving the cursor over a button on a page, the folder is automatically expanded, allowing the file to be dropped onto an item within the opened menu.
If a file is dropped onto an empty area of the menu or onto an item that does not accept drag and drop (such as a document file), the file is copied or moved to the folder currently being expanded as a menu.
If a folder item is dragged and dropped with the right mouse button, an operation selection menu is displayed.
The behavior for copying or moving files dropped into a menu follows Explorer specifications. The option When files are dropped onto a folder button in the Folder options tab does not apply.
To cancel a drag-and-drop operation, press the Esc key or drop the item outside of any buttons in the CLaunch window.
Menu options
By specifying Menu Options (the fourth input field from the top) in the folder item properties, you can change the behavior of the menu display.
