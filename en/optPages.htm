Pages (Options Dialog Box)
Page
Selects the page to configure.
Page Name
Specifies the page name displayed on the page tab.

Auto-reset of Scroll Position
Automatically resets the button scroll position at the selected timing.
This is useful if infrequently used items are placed outside the visible area.

Tab Designs
Displays previews of tabs for each state.
Click a preview image to open the corresponding Tab Design dialog for that state.
Different dialog boxes are displayed depending on whether you are using the Classic skin or the Modern skin.

Button Designs
Displays previews of buttons for each state.
Click a preview image to open the corresponding Button Design dialog for that state.
Different dialog boxes are displayed depending on whether you are using the Classic skin or the Modern skin.

Copy these designs to all pages
Applies the design of the currently displayed page to all existing pages.
Save these designs as the default for new pages
Sets the design of the currently displayed page as the default design for newly created pages.
