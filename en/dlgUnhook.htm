Auto-unhook Settings dialog
This dialog appears when you click the Auto-unhook Settings button in the Item Properties dialog for a program or document item.


The following settings apply only to the item for which this dialog was opened.
Unhook before launching this program
Automatically performs Unhook before launching the program registered to the item.
If the program is launched as a Chain Launch Item, this option is ignored.
Automatically resume the hook when the launched program exits*1
Monitors the launched program and automatically resumes the hook when it exits.
This option is ignored if the program is launched with administrator privileges.
Automatically resume the hook when the specified program exits*1
Monitors a user-specified program and automatically resumes the hook when it exits.

*1 If both hook-resume options are enabled, the hook will resume when either program exits.


The following settings are common settings that affect all registered items.
Delay between unhooking and launching the program
After unhooking, the specified delay time is applied before launching the registered program.
Resume the hook when CLaunch appears
When manually resuming the hook, you do not need to reselect Unhook from the Main Menu; the hook will automatically resume when the CLaunch window appears.
While unhooked, you can display the CLaunch window by clicking the task tray icon or by Launching Multiple Instances of CLaunch.
