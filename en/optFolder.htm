Folders (Options dialog)
Default settings when registering folders
Select the default folder open method that will be applied when you register a folder.
You can also change the folder open method after registration from the Item Properties dialog.
  • Open
    Opens the folder in the system default folder window.
  • Open in Explorer
    Opens the folder in Explorer with the navigation pane visible.
  • Open as Menu
    Opens the folder contents as a Folder Contents Menu.

When files are dropped onto a folder button
Select the action performed when you drag and drop files onto a folder item.
If Copy is selected, dragging and dropping with the mouse alone copies files, and dragging and dropping while holding the Shift key moves files.
If Move is selected, dragging and dropping with the mouse alone moves files, and dragging and dropping while holding the Ctrl key copies files.
If System Default is selected, files are moved when the source and destination are on the same drive, and copied when they are on different drives. In either case, holding Shift forces move, and holding Ctrl forces copy.

File manager
Specifies an alternative file manager to launch instead of Explorer when opening folders.
Enable Use an alternative file manager, and enter the path to the program.

Parameters
Specify common parameters to pass to the file manager via its command line.
Parameters are concatenated as follows:

  • For normal folders
    [Item-specific parameters] [Common parameters] [Folder path]
  • For special folders
    [Item-specific parameters] [Common parameters]
