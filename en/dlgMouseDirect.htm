Direct Mouse Launch dialog
This dialog box appears when you click the Assign button of Mouse Launch in the Item Properties dialog.
It allows you to launch items directly using mouse operations without displaying the CLaunch window.
Clicks
Configure launching the selected item using Click Functions.
Click Single, Double, Triple, Wheel, or Hold at the top to display the corresponding list of click operations.
Click the [+] icon next to an operation to display its valid range. Checking the box next to a range enables launching the item using that operation.
If you want it to launch regardless of cursor location, enable the option equivalent to Anywhere.
If the desired operation is already assigned to displaying CLaunch or launching another item, clear that assignment first, then assign it again.
The delay before an item assigned to a press-and-hold operation launches after pressing a mouse button can be changed in the Press-and-hold delay setting of the Clicks tab in the Options dialog.
Edges
Configure launching the selected item using Edge Functions.
Checking this row launches the item when the cursor reaches the edge of the screen.
Checking this row launches the item when the cursor leaves the screen edge after touching it.
The delay before the item launches after the cursor reaches the screen edge can be changed in the Delay before launching the items setting of the Edges tab in the Options dialog.
Circles
Configure launching the selected item using Circle Functions.
The number of rotations and detection sensitivity for circle operations can be configured in the Circles tab of the Options dialog.
