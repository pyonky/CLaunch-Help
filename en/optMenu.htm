Menus (Options dialog)
Limits
Sets various limits used when displaying submenus and Folder Contents Menus.
In addition to these settings, you can also set item-specific limits under Menu Options in the Item Properties dialog.
For all items except Number of Columns, specifying 0 means unlimited.

Number of Items (for folder contents menus only)
Limits the number of files displayed in a folder contents menu.
Reducing this value can shorten the time it takes for the menu to appear.

Column Width
Limits the width of each menu column.
If a file name does not fit within the specified width, the end of the name will be truncated.

Number of Rows
Limits the maximum number of rows in a single column.

Number of Columns
Specifies the maximum number of columns in a menu.
If you set this to 2 or higher and the number of files exceeds Number of Rows, additional columns will be displayed.

Delay before auto-open menus on mouseover
Specifies, in milliseconds, how long the cursor must remain on a button before the menu opens automatically.

When an item is set to Auto-open
The delay applied to items where Auto-open menu on mouseover is enabled under Methods to open folders in the Item Properties dialog.

When files are dragged
The delay before auto-open begins after the cursor is moved onto a button while dragging files.

Scrolling
If the total number of menu items exceeds the visible area, hidden items can be accessed by scrolling.

Auto-scroll Start Delay
Specifies, in milliseconds, the delay before auto-scrolling starts after you press a scroll button.
This setting also applies to auto-scrolling triggered by mouseover.

Auto-scroll Interval
Specifies, in milliseconds, the interval for auto-scrolling while you keep a scroll button pressed.
This setting also applies to auto-scrolling triggered by mouseover.

Scroll Lines
The number of lines scrolled per operation when the menu is shown in a single column.
When the menu is shown in multiple columns, it scrolls horizontally by one column regardless of this setting.

Scroll automatically on mouseover of the scroll buttons
Starts scrolling automatically when you move the cursor onto the menu’s scroll buttons.
While dragging files, auto-scrolling by mouseover is enabled regardless of this setting.

Display settings
Icon Size
Specifies the display size of menu icons.

Use large icons
For submenus, uses the icon images that are already obtained for button display.
For folder contents menus, uses 32×32 pixel icons.

Show tooltips
Shows tooltips when you move the cursor over menu items.
You can also toggle tooltips on/off using shortcut keys after a menu is displayed.

Load icons in the background (for folder contents menus only)
Retrieves and displays icons progressively after the menu is shown.
This can slightly shorten the time it takes for the menu to appear.

Menu font
Changes the font used for menu items.

* Icon size, use large icons, and menu font settings also apply to the result items in the Search Window.

Keyboard operations
Item selection timeout when typing an item name
Specifies, in milliseconds, the maximum interval allowed between keystrokes when selecting a menu item by typing its name.

Operate the menu under the cursor using the keyboard
Allows you to operate the menu under the cursor using the keyboard, even when a child menu is open.
If this option is disabled, keyboard operations always target the deepest (lowest-level) menu.
Even when enabled, selecting items with the cursor keys and opening/closing child menus still always targets the deepest menu.
After you perform any of those selection operations with the keyboard, all keyboard operations will switch back to the deepest menu until you select an item with the mouse again.
