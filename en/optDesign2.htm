Design 2 (Options Dialog Box)
Tabs
Tab Size
Sets the tab size. If you set it to 0, tabs are hidden.

Fixed width
Fixes the tab width to the value specified in Tab Size.
If this option is unchecked, the tab width is adjusted automatically to fit the page name, and the value of Tab Size becomes the minimum width.

Multi-line tabs
When there are too many pages to fit all tabs in one row, tabs are displayed on multiple rows.

Multi-line text
When Fixed width is enabled, long page names are wrapped and displayed on multiple lines.

Left-aligned text
Displays page names left-aligned. If this option is unchecked, text is centered.

Adjust Text Position
Adjusts the page name position.

Active Tab Font, Inactive Tab Font
Changes the tab fonts.
The active tab is the tab for the currently displayed page. Inactive tabs are all other tabs.

Buttons
Button Size
Sets the button size.

Source Icon Size
Sets the icon size (in pixels) to retrieve from the source file.
If the source file does not contain an icon of the specified size, the closest available size is used.

Icon Display Size
Sets the icon size displayed on the button.
If you specify a value different from Source Icon Size, the icon is scaled up or down.

Icon Position
Selects where the icon is displayed within the button.
If you select Top, the icon is shown at the top and the text is placed below it.
If you select Left, the icon is shown on the left and the text is placed to the right of it.

Adjust Icon Position
Adjusts the icon position.
The value on the side adjacent to the text becomes the spacing between the icon and the text.

Show text
If this option is unchecked, button text is hidden and the icon is vertically centered in the button.

Single-line text
Displays long text on a single line without wrapping. Text is also vertically centered in the text area.

Left-aligned text
Displays text left-aligned. If this option is unchecked, text is centered.

Adjust Text Position
Adjusts the text position.
The value on the side adjacent to the icon becomes the spacing between the text and the icon.

Font
Changes the button font.

View Mode
Separate designs per View Mode
If this option is enabled, you can use different design settings for View Mode 1 and View Mode 2, and button layouts are saved separately for each View Mode.
If this option is disabled, design settings and button layouts are shared between both View Modes. Only the window size (number of buttons horizontally and vertically) and the search window position are saved separately.
If you change this option from enabled to disabled, all design settings and button layouts will be overwritten with the settings of the current View Mode, and the settings of the other View Mode will be lost.
If you may want to restore the original design settings later, back up your settings before disabling this option.

Item Fill Direction
Selects the direction in which items are placed.
This option affects the behavior when performing the following operations.
  • The fill direction used when registering multiple files by drag-and-drop
  • The shift direction when items are moved between buttons or a empty button is inserted, causing items after the insertion point to shift backward
  • The shift direction when a empty button is deleted, causing items after the deletion point to shift forward
