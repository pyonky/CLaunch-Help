Settings to disable for specific applications dialog
This dialog appears when you click the Configure button for Settings to disable for specific applications in the Disabling tab of the Options dialog.
You can configure selected functions to be disabled while the specified application is activated.

When a blank row in the upper list is selected, entering a condition in Window Title, Window Class, or File Name will automatically register it as a target to disable.
Use the check boxes in the lower area to select the functions you want to disable while the target is activated.
Click the Existing Windows button to show a list of running applications. Select the target application and click OK to retrieve its information.
If you cannot find the target application in the list, click the Window Finder button and then click anywhere in the target application's window to automatically select the corresponding item in the list.
If multiple conditions are set, only windows that match all conditions will be targeted, so delete unnecessary conditions and leave them blank.
To add another application as a target, reselect a blank row and then enter new conditions.
Window Title is evaluated by partial match, and Window Class is evaluated by prefix match.
The behavior of File Name differs depending on whether you specify a full path, only a file name, or a folder path.
If you specify a full path or only a file name, it is evaluated by exact match. If you specify a folder path, it is evaluated by prefix match, so all programs located under the specified folder will be targeted for disabling.

Example 1)If you specify "C:\games\main\abc.exe", "C:\games\main\abc.exe" will be targeted, and "C:\games\sub\abc.exe" will be excluded.
Example 2)If you specify "abc.exe", both "C:\games\main\abc.exe" and "C:\games\sub\abc.exe" will be targeted.
Example 3)If you specify "C:\games", "C:\games\main\abc.exe", "C:\games\sub\abc.exe", and "C:\games\sub\xyz.exe" will all be targeted.
