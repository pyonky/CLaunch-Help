Switching View Modes
Switching the view mode allows you to instantly change the number of visible buttons.
For example, you can arrange less frequently used items so that they are hidden outside the visible area in View Mode 1, and then switch to View Mode 2 to display all items.
Depending on the option settings, you can also switch between the default panel layout and a list layout with horizontally arranged buttons.

To switch the view mode, select View Mode 1 or View Mode 2 from the Main Menu, or double-click the title bar.
If the current skin supports it, a view mode button is displayed, and you can also switch view modes by clicking it.
By default, the following settings are stored separately for each view mode: the window size (number of visible buttons), the width and position of the search window, and the number of scroll lines per page.
When Separate designs per View Mode is enabled in the Design 2 tab of the Options dialog, the sizes of tabs and buttons, button content settings, icon sizes, fonts, and other design-related settings are also stored separately for each view mode.
If you hide less frequently used items in View Mode 1, it is convenient to enable Switch to View Mode 1 after an item is launched in the Other tab of the Options dialog.
