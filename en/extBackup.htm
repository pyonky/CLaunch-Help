Backing Up Settings
Backing Up All Settings
You can back up item registration data, option settings, and design settings all at once.
It is recommended to back up your settings regularly to prepare for unexpected issues.
This is also useful when migrating your environment to another computer.
To back up settings, select Maintenance > Backup Settings from the Main Menu, then specify the destination for the backup file.
Although a single INI file is created, it contains all settings data, including design settings.
Settings backups can also be performed automatically on a regular basis. Automatic backup can be configured in the Other tab of the Options dialog.
Backing Up Design Settings
You can back up only the design settings separately.
To back up design settings, select Maintenance > Backup Design from the Main Menu, then specify the destination for the backup file.
If you add pages after backing up design settings and then restore the design from the backup, the design of the newly added pages will be reset to the default settings.
Restoring from a Backup
You can restore settings from a backup file using the following procedure.
  1. If the backup file name is not CLaunch.ini, rename it to CLaunch.ini. (For a design settings backup, rename the file to Design.ini.)
  2. From the Main Menu, select Maintenance > Open Data Folder to open the folder where settings data is stored.
  3. From the Main Menu, select Exit to terminate CLaunch.
  4. Overwrite the existing file in the data folder with the backup file.
  5. Start CLaunch.
