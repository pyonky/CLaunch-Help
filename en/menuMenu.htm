General Menu of the Folder Contents Menu
Displayed when you right-click an empty area in a folder contents menu, or Shift + right-click a menu item.
Sort by Name, Sort by Size, Sort by Extension, Sort by Date
Changes the sorting order of menu items.
Selecting the same sort option again toggles between ascending and descending order.

Tooltips
Toggles display of tooltips shown when the cursor remains over a menu item.

Select All
Selects all items in the menu.
Deselect All
Clears the selection of selected items.

Paste
Pastes files from the clipboard into the folder currently displayed as a menu.
This function is available only when files have been copied (Ctrl + C) or cut (Ctrl + X) beforehand.
Create Folder
Creates a new folder under the folder currently displayed as a menu.

Open This Folder
Opens the folder currently displayed as a menu in Explorer.
Show Parent Folder
Displays, as a menu, the parent folder one level above the top-level folder originally expanded.

Close
Closes the current menu and all menus in lower hierarchy levels.
