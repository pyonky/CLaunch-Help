Temporarily Disabling CLaunch Functions
When CLaunch becomes intrusive, such as while playing games or watching videos, its functions can be temporarily disabled using the methods described below.
Restricted Mode
Restricted Mode allows you to disable CLaunch functions at any time.
The functions disabled while Restricted Mode is active can be selected in Functions to disable in Restricted Mode on the Disabling tab of the Options dialog.
To enter Restricted Mode, select Restricted Mode from the Main Menu, or double-click the task tray icon .
When Restricted Mode is activated, the task tray icon changes to .
To exit Restricted Mode, select Restricted Mode again from the Main Menu, or double-click the task tray icon.
If the task tray icon is hidden and Restricted Mode cannot be turned off, you can forcibly display the window by launching CLaunch.exe again.
By registering the Restricted Mode special item to a button and assigning a hotkey in Item Properties, you can toggle Restricted Mode using a hotkey. In this case, be sure to uncheck Hotkeys in Functions to disable in Restricted Mode on the Disabling tab of the Options dialog.
Disabling While a Full-screen Application Is Active
Selected functions can be automatically disabled while a full-screen application is active.
The functions to disable can be selected in Functions to disable while a full-screen app is active on the Disabling tab of the Options dialog.

* Depending on the application, full-screen status may not be detected correctly.
Disabling When Specific Applications Are Active
Selected functions can be automatically disabled when specified applications are active.
The functions to disable can be selected in Settings to disable for specific applications on the Disabling tab of the Options dialog.
Unhooking
If Restricted Mode does not work properly with certain applications, such as games, you can forcibly disable CLaunch functions by temporarily unhooking them.
Unhooking may also help resolve issues such as reduced game performance, error messages at startup, or false detection as a cheat tool, by unhooking before launching the application.
While unhooked, CLaunch cannot be shown using mouse actions or hotkeys, and registered items cannot be launched directly. After finishing with the affected application, be sure to re-enable hooks.
To unhook, select Unhook from the Main Menu. While unhooked, the task tray icon changes to .
Selecting Unhook again from the Main Menu will re-enable hooks.
If the task tray icon is hidden and hooks cannot be re-enabled, you can forcibly display the window by launching CLaunch.exe again.
By configuring Auto-unhook Settings in Item Properties, hooks can be automatically disabled before launching a program.
You can also configure hooks to be automatically re-enabled when the launched program or specified program exits.
