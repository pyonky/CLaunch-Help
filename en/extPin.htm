Pin the CLaunch Window
Pins the CLaunch window so that it remains visible on the screen.
This is useful when repeatedly registering items by drag-and-drop or launching multiple items in succession.

To enable Pin mode, select Pin from the Main Menu or middle-click the title bar.
While pinned, the pin indicator on the title bar is shown as active.

  

To disable Pin mode, select Pin again from the Main Menu or middle-click the title bar.
(The pin state can also be toggled by clicking the pin indicator on the title bar.)
If both Always on Top and Only while pinned are enabled in the Display tab of the Options dialog, CLaunch will stay on top of other windows while pinned.
