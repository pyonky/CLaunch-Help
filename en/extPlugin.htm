Plugins
By adding plugins, you can extend CLaunch with functionality that is not provided by default.

To add a plugin, copy the plugin module (DLL file) to the Plugins folder under the CLaunch installation directory, then register it from the Plugins tab in the Options dialog.
Select the desired plugin from the list on the Plugins tab in the Options dialog, then click the Plugin Options button to configure plugin-specific settings.
For plugins that can be registered to buttons as plugin items, open Item Properties after registering the button, then click the Item Properties button in Plugin Item Options to configure settings specific to the plugin item.
The website provides currently available plugins. It also includes a link to the Plugin Development Kit published on GitHub.
