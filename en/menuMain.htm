Main Menu
It is displayed when you right-click the task tray icon or the title bar.
Pin
Temporarily pins the CLaunch window so it does not disappear from the screen.
Pin
Restricted Mode
Temporarily disables the functions selected in the Functions to disable in Restricted Mode setting in the Disabling tab of the Options dialog.
Restricted Mode
Search Items
Displays the search window and lets you search registered items by entering keywords.
Item Search

View Mode 1, View Mode 2
Switches to the selected View Mode.

Unhook
Temporarily unhooks CLaunch.
Add Page
Adds a new page. This is the same as Add Page in the Page Menu.
Maintenance
Hotkey List
Displays a list of hotkeys assigned to all registered items.
Click the Copy to Clipboard button to copy the hotkey list to the clipboard as tab-delimited text.

Backup Settings
Backs up item registration data, option settings, and design settings all together.
Backing Up Settings

Backup Design
Backs up only the design settings separately.

Open the Data Folder
Opens the folder where configuration files are stored.

Convert to Relative Path
Batch-converts all registered items from absolute paths to relative paths.
Conversion Between Relative and Absolute Paths

Convert to Absolute Path
Batch-converts all registered items from relative paths to absolute paths.

Update Icons
Batch-updates the icons of all registered items.
Updating Icon Images
Skins
Displays the Skins tab of the Options dialog.
* This menu item is disabled on operating systems earlier than Windows 2000.
Options
Displays the CLaunch Options dialog where you can configure general settings for CLaunch.

Languages
Displays the Choose Language dialog.
The language list shows the languages provided by the language modules placed in the Languages folder.
Select a language from the list and click the OK button to switch the interface language.
* This menu item is disabled in the ANSI version.
CLaunch Help
Displays the help you are currently viewing.
About CLaunch
Shows version information for CLaunch.

Exit
Exits CLaunch and stops it from running in the background.
