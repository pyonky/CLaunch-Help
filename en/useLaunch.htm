Launching Items
Clicking a button with a registered item launches the assigned item.
When using the keyboard, select an item with the cursor keys and press the Enter key to launch it.

If you left-click a button while holding down the Ctrl key, or press the Space key while the button is selected, the item is launched in Continuous Launch Mode.
In Continuous Launch Mode, the CLaunch window remains visible in the foreground after launching an item, allowing you to launch multiple items consecutively.
This mode is automatically canceled when CLaunch is hidden or when an item is launched normally by a single click or by pressing the Enter key.
If a Shortcut Key is assigned to an item, the item can be launched by pressing the assigned key while the page containing that item is displayed.
If a Hotkey is assigned to an item, the item can be launched by pressing the assigned key even when the CLaunch window is not visible.
If Mouse Launch is assigned to an item, the item can be launched using the assigned mouse action even when the CLaunch window is not visible.
If Chain Launch Items is assigned to an item, multiple items can be launched simultaneously.
If a document file is registered, the application associated with that file is launched.
If a URL is registered, the default web browser is launched.
For buttons with a registered folder, you can choose one of three open methods from the Item Menu: Open, Open in Explorer or Open as Menu.
