Register Item dialog
Displayed when you select Register Item from the Empty Button Menu or from the Item Menu of a submenu.
File / URL
Specify the path of the file or folder to register to a button or submenu, or enter a URL.

You can use relative paths from the CLaunch installation folder and environment variables in the path.
If a shortcut is specified, checking Convert shortcut to linked target registers the actual target file. If unchecked, the shortcut itself is registered. (Folders are always converted to their target.)
If you register an email address in the format "mailto:abc@xyz.com", the button will create a new email addressed to that recipient.
Using the Existing Windows button allows you to easily register a currently running application. (Windows 2000 or later)
  1. Start the application you want to register.
  2. Click the Existing Windows button.
  3. Click any area of the application window.
  4. Confirm that the path is entered in the File / URL field, then click OK.
Special Folders
Register folders such as Desktop or Control Panel whose original icons cannot be easily located.
Special Items
Assign special functions such as CLaunch control, power control, or plugin items to a button.
