Item Search
When the number of registered items increases, or when you want to quickly launch an item located deep inside submenus, you can search for items by entering keywords.
Select Search Items from the Main Menu, or press Ctrl + F on the keyboard to display the search window.

While the search window is displayed, enter a string contained in the target item name. Matching items will be listed. Select the desired item using the arrow keys and press Enter to launch it.
Items whose search word appears closer to the beginning of the item name are shown higher in the list.
While the search window is displayed, selecting Search Items again from the Main Menu or pressing Ctrl + F will close the search window.
If the current skin supports it, a search button is displayed, and clicking it also toggles the search window on and off.
While the search window is displayed, pressing the Tab key switches the keyboard input focus alternately between the search window and the main window.
Icons and text shown in the search window follow the submenu display settings configured in the Menus tab of the Options dialog.
The position and width of the search window can be adjusted by dragging it with the mouse.
When you open the Item Properties of an item listed in the search results, the display switches to the page that contains that item.
To automatically close the search window after launching an item, enable Close search window after an item is launched in the Other tab of the Options dialog.
When Include tooltip text in item search in the Other tab of the Options dialog is enabled, items whose Tooltip text contains the search word will also be included in the search results.
When Start searching by typing a character key in the main window is enabled in the Keyboard tab of the Options dialog, typing a character key while the CLaunch window is displayed will automatically show the search window and start searching immediately, even if the search window is not currently visible. (Basic operations and item-specific shortcut keys assigned to single character keys take priority.)
When this option is enabled, the search window will close automatically when all search text is deleted.
