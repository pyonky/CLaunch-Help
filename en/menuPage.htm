Page Menu
Displayed when you right-click a page tab.
Add Page
Adds a new page. Same as Add Page in the Main Menu.
Delete Page
Deletes the currently displayed page.

Sort Items
Sorts the registered items in the page by name.
Locked buttons are excluded from sorting.
Launch all items in the page
Launches all registered items in the page at once.
Additional Functions special items and items inside submenus are not included.

Properties
Displays the Pages tab of the Options dialog, where you can configure page names and per-page design settings.
