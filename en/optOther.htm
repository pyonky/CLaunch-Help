Other (Options Dialog Box)
Backup the settings periodically
Automatically backs up CLaunch settings at regular intervals.
Specify the backup file path and the number of days between backups.

Single user mode
All logged-in users share item registration data and option settings.
If you carry CLaunch on a USB drive, enabling this option allows you to use the same settings even when the login user name changes on different computers.
Switch pages when you mouseover the tabs
Switches pages simply by hovering the cursor over a page tab.
To change the delay before switching pages, adjust Delay before switching pages automatically in the Edges tab of the Options dialog.
Loop page switching
When you switch pages on the first or last page, the page selection loops to the opposite end.
Save Pinned status
Keeps the pinned state even when CLaunch is hidden.
If CLaunch is exited while pinned, it starts in the pinned state the next time it is launched.
Save Restricted Mode status
If CLaunch is exited while in Restricted Mode, it starts in Restricted Mode the next time it is launched.
Include tooltip text in item search
When searching for items, items whose Tooltip text contains the search word will also be included in the search results.
Close search window after an item is launched
Automatically closes the search window after launching an item while the search window is open.
Switch to View Mode 1 after an item is launched
Automatically switches to View Mode 1 after launching an item in View Mode 2.
This is useful if infrequently used items are placed outside the visible area in View Mode 1.
Unpin after an item is launched by drag-and-drop
After launching a program by drag-and-drop onto a registered item, automatically unpins CLaunch and hides it.
Register items with a relative path
Automatically converts the file path and working folder path to relative paths when registering a new item.
Show button properties after an item is registered
Displays the item properties dialog after registering a new item.
This does not apply when multiple files are registered at once by drag-and-drop.
Don't create an empty button when an item is deleted
When deleting a registered item, shifts subsequent buttons forward to prevent leaving an empty button.
Free unused physical memory
Releases physical memory used by CLaunch as much as possible.
If you experience sluggish performance, disabling this option may improve responsiveness.
CLaunch process priority
Specifies the processing priority of CLaunch within the system.
Selecting Realtime or High may reduce overall system performance.
