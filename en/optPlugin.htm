Plugins (Options dialog)
Select the plugins that add features to CLaunch.
To enable a plugin, check the box to the left of its name in the list. (Plugins listed here do not work unless they are enabled.)
About This Plugin
Shows version information, copyright information, and other details for the plugin selected in the plugin list.
Plugin Options
Opens the settings dialog for the plugin selected in the plugin list.
Open the Plugins Folder
Opens the folder that contains the plugin modules (DLL files).
