Updating Icon Images
CLaunch caches icon images so that the program can start quickly even when many items are registered. For this reason, if a registered program is updated and its icon changes, or if an associated icon changes, the new icon may not be reflected and the old icon may continue to be displayed.
In such cases, performing a batch icon refresh will update the icons to the latest versions.

To perform a batch icon refresh, select Main Menu > Maintenance > Update Icons.
A batch icon refresh may take some time, and CLaunch will be temporarily unavailable during the process. (While the refresh is in progress, the tray icon continues animating.)
