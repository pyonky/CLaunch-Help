Additional Functions
Recent Items
Displays a menu listing the history of items launched from CLaunch.

Last Displayed Menu
Redisplays the last displayed Submenu or Folder Contents Menu.

Clipboard Folder Menu
Opens the folder whose path is copied to the clipboard as a menu.

Launch Screen Saver
Starts the screen saver selected in Windows settings.

Suspend Screen Saver
Suspends screen saver activation and display power saving mode.

Power Off Display
Turns off the screen. It will restore when you operate the keyboard or mouse.

Windows Shutdown Menu
