Limitations
CLaunch cannot be installed in an extremely deep folder hierarchy. Specifically, if the path names of data files under the installation folder, or configuration files such as skins, exceed 260 characters (260 bytes for the ANSI version), the program will not operate correctly.
Similarly, some option items that handle path names, such as the executable file name for Settings to disable for specific applications, the executable file name of a file manager, and tray icon file names, are also limited to a maximum length of 260 characters.
However, there is no limitation on the length of strings set in the Item Properties dialog, such as the path names of registered items.

There are the following restrictions when registering UWP apps (such as Microsoft Store apps and Windows Settings).
  • Application-specific icons cannot be obtained. (There are some exceptions.)
  • The privilege level cannot be changed (for example, Run as administrator is not available).
  • Depending on the application, settings such as Window State, Priority, and Action when the program is already running specified in the item properties may not function properly.
Mouse
To use the mouse wheel assignment feature, a mouse with a wheel is required. To use the tilt wheel assignment feature, a mouse with a tilt wheel is required.
To use the wheel click assignment feature, assign Middle Button (or Auto Scroll, depending on the driver) to the wheel button in the mouse driver settings.
To use the side button assignment feature, use a mouse with four or more buttons, and assign Back to side button 1 and Forward to side button 2.
