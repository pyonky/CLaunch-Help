Tab / Button Design Dialog Box (Modern skins)
This dialog box appears when a skin other than Classic is applied and you click the preview image of a tab or button shown in the Pages tab of the Options dialog.
Design Pattern
If the applied skin provides multiple design patterns for tabs and buttons, select the background design here.
Text Color
Sets the text color displayed on tabs and buttons.
If a text shadow is selected, you can also change the shadow color.

Shadow
Adds a shadow to the text. If you select Outline, drawing performance may decrease.

Shadow Width
If the shadow type is Upper Left or Lower Right, the value specifies the distance between the text and the shadow. If it is Outline, the value specifies the outline width.

Shadow Alpha
Specifies the transparency of the text shadow. (0%: Opaque - 100%: Transparent)
