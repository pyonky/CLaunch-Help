Mouse Operations
By default, the following mouse actions are assigned to each function.
Mouse assignments can be changed in the Mouse tab of the Options dialog.
Common Mouse Operations in the CLaunch Window
Rotate the wheel to switch pages.
Rotate the wheel while holding the right button to scroll vertically within the page, or rotate the wheel while pressing it to scroll horizontally. (Only when items extend outside the visible area)
Tilt the wheel to the left to jump to the first page, or to the right to jump to the last page.
Click the X1 button to jump to the first page.
If the first page is already displayed, jump to the last page.
Click the X2 button to jump to the last page.
If the last page is already displayed, jump to the first page.
Mouse Operations on Buttons
Hold the left button and right-click to open the folder. *1
Wheel-click to Open Folder as Menu. *1
Ctrl + left-click launches the item in Continuous Launch Mode.
Alt + left-click opens the parent folder. *1
Shift + right-click shows the shell context menu.
Ctrl + right-click locks the button.
Alt + right-click shows the Item Properties dialog.
Press and hold the Recycle Bin button for one second to empty the Recycle Bin.
Press and hold a power control item button for one second to start forced shutdown without confirmation.
Because shutdown begins without waiting for applications to respond, unsaved data may be lost. Use at your own risk.
Press and hold buttons other than Recycle Bin or power control items for one second to open the parent folder as a menu. *1

*1 Difference between Open Folder / Open Folder as Menu
and Open Parent Folder / Open Parent Folder as Menu
For buttons registered with a folder, Open Folder and Open Folder as Menu open the registered folder itself. Open Parent Folder and Open Parent Folder as Menu open the parent folder one level above.
For non-folder items, both operations behave the same.
Folder Contents Menus opened by Open Parent Folder as Menu also display hidden files.
Mouse Operations on the Title Bar
Double-click to switch the View Mode.
Wheel-click to toggle pin on or off.
Alt + right-click shows the CLaunch Options dialog.
Mouse Operations on the Tray Icon
Double-click to toggle Restricted Mode on or off.
Wheel-click shows the CLaunch Options dialog.

* If an assignment to Click Functions conflicts with tray icon mouse operations, the Click Function assignment takes priority.
Mouse Operations on Submenus and Folder Contents Menus
Wheel-click closes the clicked menu and its child menus.
Rotate the wheel to scroll the menu. (Rotating the wheel over scroll buttons performs page scrolling.)
Rotate the wheel while holding the right button to scroll by page.
Click the X1 button to scroll to the previous page.
Click the X2 button to scroll to the next page.
Tilt the wheel left to select the first item, or right to select the last item in the menu.
Press and hold a menu item to rename the item (or file name).
Ctrl + left-click selects multiple items.
Shift + left-click selects a range of items.
Alt + right-click shows item (or file) properties.
Shift + right-click shows the shell context menu for submenus, or the General Menu for Folder Contents Menus.
Hold the left button and right-click to open the selected folder, or the parent folder in Explorer if the item is not a folder.
