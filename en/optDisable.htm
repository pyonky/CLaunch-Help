Disabling (Options dialog)
Lets you automatically disable CLaunch functions depending on which program is active.
While any CLaunch dialog is displayed, click functions and hotkeys are disabled regardless of the settings below.
Functions to disable while CLaunch is active
Select which functions to disable while the CLaunch window is active.
If a hotkey uses the same key as basic operation shortcut keys or an item shortcut key, disable hotkeys while CLaunch is active to avoid conflicts.
If a click function uses the same operation as a mouse operation assigned to a basic operation, disable click functions while CLaunch is active to avoid conflicts.
Functions to disable in Restricted Mode
Select which functions to disable in Restricted Mode.
If the task tray icon is hidden and you disable all functions in Restricted Mode, you may have no way to show CLaunch and you will not be able to exit Restricted Mode.
Functions to disable while a full-screen app is active
Lets you automatically disable selected functions while a full-screen program such as a game or video player is active.
Depending on the program, full-screen state may not be detected correctly.
Settings to disable for specific applications
Lets you automatically disable selected functions while specified applications are active.
Click the Configure button to open the Settings to disable for specific applications dialog.
