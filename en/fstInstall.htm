Installation and Uninstallation
Automatic Installation
Double-click the downloaded self-extracting archive to start Setup.exe and begin the installation. When the installation is complete, CLaunch starts automatically and a tray icon appears in the system tray.
If CLaunch is registered in Startup, it will automatically run when Windows starts the next time.
If you check Startup with ClAdmin when registering CLaunch in Startup, mouse actions and hotkeys assigned to displaying CLaunch or launching applications will also work on applications running with administrator privileges.
Manual Installation
Extract the downloaded ZIP archive to any folder and run CLaunch.exe to start using it immediately.
If you want to install CLaunch on removable media such as a USB flash drive, use this method.
When manually installing to a fixed disk, create a shortcut to CLaunch.exe and register it in Startup if necessary.
Updating
When updating from a previous version, be sure to perform an overwrite installation.
If you uninstall a previous version, all registered items and configuration data will be deleted.
Uninstallation
If CLaunch was installed using the automatic installer, it is registered in Programs and Features in Control Panel. Select CLaunch from the list and choose Uninstall or change to remove it automatically.
If CLaunch was installed manually, delete the extracted files and the data files created during execution by removing the entire folder.
Data Folder Location
The location of the data folder depends on the CLaunch installation folder.

If installed in a folder where writing is restricted by UAC (such as C:\Program Files):
 Multi-user mode (default)    -> C:\Users\Username\AppData\Roaming\CLaunch
 Single-user mode (optional) -> C:\ProgramData\CLaunch

If installed in a folder without write restrictions:
 The Data folder under the CLaunch installation directory
Registry
CLaunch does not use the registry.
However, when using automatic installation, uninstallation information is registered at the following location:

[HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Uninstall]

This information is removed during uninstallation.
