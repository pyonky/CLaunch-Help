Mouse (Options dialog)
Mouse Operations
Allows you to change the assignments for mouse operations.
Select a target from the Targets list, choose the mouse operation you want to assign from the Operations list, and then select the function to assign from the Assigned Function list.

Empty Recycle Bin by ... on the Recycle Bin button
Performs the Windows Recycle Bin empty operation by the specified mouse action on a button registered as the Recycle Bin.

Force Shutdown by ... on the Power Control button
Performs a forced shutdown by the specified mouse action on a button registered as a power control item, without displaying a confirmation message.
Because forced shutdown begins without waiting for applications to respond, unsaved data may be lost. Use at your own risk.

Press-and-hold delay
Specifies the waiting time between pressing the left mouse button on a button or menu item and executing the function assigned to press-and-hold.

Drag-and-drop

Button Move Start Operation
Selects the modifier key used when moving buttons by drag-and-drop.
By default, buttons can be moved using drag-and-drop with the mouse alone. Specifying a modifier key helps prevent unintended button movement.
This setting does not apply to moving submenu items. Submenu items can be moved only by drag-and-drop with the mouse alone.

OLE Drag-and-drop Start Operation
Selects the operation used to start OLE drag-and-drop.
Right-button drag always performs OLE drag-and-drop regardless of this setting.
This setting does not apply to OLE drag operations for submenu items. Submenu items support OLE drag-and-drop only with the right mouse button.
