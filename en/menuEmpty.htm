Empty Button Menu
Displayed when you right-click an unregistered button.
Register Item
Displays the Register Item dialog to register a new item.
Create Submenu
Creates a new submenu.

Insert Empty Button
Inserts an empty button by shifting registered items after the target position backward.
Delete Empty Button
Deletes the empty button at the target position and shifts subsequent registered items forward.
Row/Column Operations
Allows insertion or deletion of empty rows or columns.
Insert Empty Row shifts rows downward from the row containing the selected button and inserts a blank row.
Insert Empty Column shifts columns rightward from the column containing the selected button and inserts a blank column.
Delete Empty Row deletes the row containing the selected button and shifts subsequent rows upward.
Delete Empty Column deletes the column containing the selected button and shifts subsequent columns leftward.

* When performing delete operations, if any button in the target row or column has a registered item, that button will not be moved.
