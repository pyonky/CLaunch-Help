Edges (Options dialog)
Edge Functions
Allows you to show CLaunch or launch registered items when the cursor is moved to the edge of the screen.
You can assign functions individually to the four corners and the four edges of the screen.

Click a button positioned around the screen preview to display the function assignment menu.
If you select Show CLaunch, the Initial Page dialog appears. Select the page you want to display first. If you select Last Displayed Page, the page that was visible before CLaunch was hidden will be displayed.
To change the initial page, select Show CLaunch again from the function assignment menu to reopen the Initial Page dialog.
If Show CLaunch is selected while mouse direct launch is assigned to an item, the current assignment will be cleared.
Mouse direct launch for individual items can be configured from Mouse Launch in the Item Properties dialog.
Only show CLaunch while a mouse button is pressed
Restricts showing CLaunch by edge functions to when the left or right mouse button is pressed.
This is useful if you want to use edge functions only while dragging and dropping files. However, CLaunch will still be shown if you move the cursor to the screen edge first and then press a mouse button.
Hide CLaunch when the cursor leaves the window
After CLaunch is shown by a edge function, it will automatically be hidden when the cursor leaves the window.

Delays
Delay before displaying CLaunch
Specifies the waiting time between moving the cursor to the edge of the screen and displaying CLaunch.

Delay before hiding CLaunch
Specifies the waiting time after the cursor leaves the window before CLaunch is hidden automatically.
This setting also applies when CLaunch is shown by Click, Circle, or Hotkey functions.

Delay before launching items
Specifies the waiting time between moving the cursor to the edge of the screen and launching the item assigned to the edge function.

Delay before switching pages automatically
Specifies the waiting time before a page is switched automatically when the cursor moves over a page tab.
This setting applies in the following situations:
  • During drag-and-drop operations
  • While moving registered items
  • When Switch pages when you mouseover the tabs in the Other options is enabled
