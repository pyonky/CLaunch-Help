Skins
By applying skins, you can change the appearance of CLaunch. (Windows 2000 or later)

To change the skin, select Skins from the Main Menu, then choose the desired skin from the displayed skin list.
If the selected skin cannot be used due to an issue, the Classic skin is automatically selected.
You can also create your own skins. Compress the created skin files into ZIP format and copy them to the Skins folder under the CLaunch installation directory, or create a subfolder for each skin under the Skins folder and copy all required files without compression.
The website provides the skin file specifications as well as links to skin distribution sites created by volunteers.
