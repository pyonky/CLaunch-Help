Keyboard Operations
By default, the following shortcut keys are assigned to each function.
Key assignments can be changed in the Keyboard tab of the Options dialog.
Page Switching
Keys used to switch the currently displayed page.
[Ctrl] + [Page Up] : Previous Page
[Ctrl] + [Page Down] : Next Page
[Ctrl] + [Home] : First Page
[Ctrl] + [End] : Last Page
[1] ... [0] : Switch to the corresponding page
Button Operations
Perform various operations on the button selected with the cursor keys.
[Left] , [Up] , [Right] , [Down] : Select Button
[Home] : Select First Button
[End] : Select Last Button
[Ctrl] + [Up] , [Ctrl] + [Down] ,
[Ctrl] + [Left] , [Ctrl] + [Right]		 : Scroll
[Enter] : Press Button
[Ctrl] + [L] : Lock Button
[Space] : Continuous Launch
[Ctrl] + [Enter] : Open Folder*1
[Shift] + [Enter] : Open Folder as Menu*1
[Insert] : Insert Empty Button
[Delete] : Delete Button
[Ctrl] + [P] : Item Properties
[Backspace] : Open Parent Folder*1
[Shift] + [Backspace] : Open Parent Folder as Menu*1
[Alt] + [Enter] : Show Shell Menu
[Ctrl] + [B] : Show Item Menu

*1 Difference between Open Folder / Open Folder as Menu
and Open Parent Folder / Open Parent Folder as Menu
For buttons registered with a folder, Open Folder and Open Folder as Menu open the registered folder itself. Open Parent Folder and Open Parent Folder as Menu open the parent folder one level above.
For non-folder items, both operations behave the same.
Folder Contents Menus opened by Open Parent Folder as Menu also display hidden files.
General Operations

[Ctrl] + [F] : Item Search

[Ctrl] + [M] : Show Main Menu

Common operations for Submenus and Folder Contents Menus
[Up] , [Down] : Select Up Item, Select Down Item
[Right] : Single-column menu : Open Child Menu
Multi-column menu : Select Right Item
[Left] : Single-column menu : Close Child Menu
Multi-column menu : Select Left Item
[Home] : Select First Item
[End] : Select Last Item
[Space] : Select Multiple Items
[Shift] + [Space] : Select Item Range
[Ctrl] + [A] : Select All
[Ctrl] + [D] : Deselect All
[Ctrl] + [Up] , [Ctrl] + [Down] ,
[Ctrl] + [Left] , [Ctrl] + [Right]		 : Scroll
[Page Up] : Page Scroll Up
[Page Down] : Page Scroll Down
[Esc] : Close All Menus
[Enter] : Open
[Delete] : Delete
[Ctrl] + [P] : Properties
[F2] : Rename
[Ctrl] + [N] : Create New Folder or Child Submenu
[Ctrl] + [Enter] : Open Folder*2
[Backspace] : Open Parent Folder as Menu*3
[Ctrl] + [T] : Toggle Tooltip
[Alt] + [Enter] : Open Shell Menu
[Ctrl] + [M] : Open General menu

*2 Open Folder behavior
If the selected item is a folder, that folder is opened. Otherwise, the parent folder is opened in Explorer.

*3 Open Parent Folder behavior
For submenus, the parent folder of the selected item is opened as a menu. For Folder Contents Menus, the parent folder of the displayed folder is opened as a menu.
Operations for Folder Contents Menus
[Ctrl] + [X] : Cut
[Ctrl] + [C] : Copy
[Ctrl] + [Shift] + [C] : Copy Path Name
[Ctrl] + [V] : Paste
