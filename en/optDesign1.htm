Design 1 (Options dialog)
Window Frame (Classic skins only)
Title Bar Height
Sets the title bar height. If you set it to 0, the title bar is hidden.

Frame Width
Sets the frame width on the left, right, and bottom edges of the window. If you set it to 0, the window frame is hidden.
Window Frame Background Color (Classic skins only)
Sets the background color of the window frame.
Select a default color set from the drop-down list on the left, or click the color sample box on the right to create a custom color.
The background pattern can be selected from gradient or solid color.
In gradient mode, clicking swaps the upper and lower colors.
Window Frame Outline Color (Classic skins only)
Sets the outline color of the window frame.
The color selected in the upper drop-down list is applied to the left and top edges, and the color selected in the lower drop-down list is applied to the right and bottom edges.
If Select a suitable color automatically is checked, colors matching the background are selected automatically.

Transparency (Windows 2000 or later)
Makes the CLaunch window semi-transparent. Use the slider to adjust the transparency of the background.

Transparent only when inactive
The window becomes semi-transparent only while CLaunch is inactive.

Caption
Select the text to display in the title bar.
  • Don't Display Caption
    Does not display any text in the title bar.
  • Display Item Name
    Displays the name of the selected item.
    When you switch pages, the page name is displayed.
  • Display Tooltip Text of Item
    Displays the tooltip text set for the selected item.
    When you switch pages, the page name is displayed.
  • Always Display the Following Text
    Always displays the same text.
    Enter the text you want to display in the edit box below.
Centered
Centers the caption within the caption area.

Multi-line
Wraps the caption text if it exceeds the width of the caption area.

Adjust Text Position
Fine-tunes the caption position.

Font
Changes the caption font.
Caption Text Color
Sets the caption text color.
If a text shadow is selected, you can also change the shadow color.

Shadow
Adds a shadow to the text.

Shadow Width
If the shadow type is Upper Left or Lower Right, the value specifies the distance between the text and the shadow. If it is Outline, the value specifies the outline width.

Shadow Alpha
Specifies the transparency of the text shadow. (0%: Opaque - 100%: Transparent)
