キーボード操作
標準では以下のショートカットキーが各機能へ割り当てられています。
ショートカットキーの割り当ては、オプションの「キーボード」で変更できます。
ページ切替え系
表示ページの切替え操作に使用するキーです。
[Ctrl] + [Page Up]： 1ページ戻る
[Ctrl] + [Page Down]： 1ページ進む
[Ctrl] + [Home]： 先頭ページへ移動
[Ctrl] + [End]： 最終ページへ移動
[1]～[0]： 対応位置のページへ移動
ボタン操作系
カーソルキーで選択したボタンに対し、各種操作を行います。
[←], [↑], [→], [↓]： ボタン選択
[Home]： 先頭のボタンを選択
[End]： 最後のボタンを選択
[Ctrl] + [↑], [Ctrl] + [↓],
[Ctrl] + [←], [Ctrl] + [→]		： スクロール
[Enter]： ボタンを押す
[Ctrl] + [L]： ボタンをロック
[Space]： 連続起動
[Ctrl] + [Enter]： フォルダーを開く※1
[Shift] + [Enter]： フォルダーをメニュー展開※1
[Insert]： 空白ボタン挿入
[Delete]： ボタン削除
[Ctrl] + [P]： アイテムのプロパティ
[Backspace]： 親フォルダーを開く※1
[Shift] + [Backspace]： 親フォルダーをメニュー展開※1
[Alt] + [Enter]： シェルのメニューを表示
[Ctrl] + [B]： アイテムメニューを表示

※1 「フォルダーを開く」と「親フォルダーを開く」、および
「フォルダーをメニュー展開」と「親フォルダーをメニュー展開」の違い
フォルダーを登録したボタンに対して、「フォルダーを開く」と「フォルダーをメニュー展開」は登録フォルダーそのものをオープンし、「親フォルダーを開く」と「親フォルダーをメニュー展開」では1階層上位のフォルダーをオープンします。
フォルダー以外の登録ボタンに対しては、どちらも同じ動作になります。
なお「親フォルダーをメニュー展開」で開いたメニューには隠し属性のファイルも表示されます。
全般操作系
[Esc]： 閉じる
[Ctrl] + [Space]： ウィンドウロック
[Ctrl] + [Q]： 制限モード
[Ctrl] + [F]： アイテム検索
[Tab]： 表示モード切替え
[Ctrl] + [O]： オプション
[F1]： ヘルプ
[Ctrl] + [M]： メインメニューを表示
[Alt] + [F4]： CLaunch終了
サブメニュー、フォルダー展開メニュー共通
[↑], [↓]： 上下アイテム選択
[→]： 1列メニューの場合   : 子メニューを開く
複数列メニューの場合 : 右アイテム選択
[←]： 1列メニューの場合   : 子メニューを閉じる
複数列メニューの場合 : 左アイテム選択
[Home]： 先頭のアイテムを選択
[End]： 最後のアイテムを選択
[Space]： 複数アイテム選択
[Shift] + [Space]： 範囲アイテム選択
[Ctrl] + [A]： 全て選択
[Ctrl] + [D]： 選択解除
[Ctrl] + [↑], [Ctrl] + [↓],
[Ctrl] + [←], [Ctrl] + [→]		： スクロール
[Page Up]： ↑ページスクロール
[Page Down]： ↓ページスクロール
[Esc]： 全てのメニューを閉じる
[Enter]： 開く
[Delete]： 削除
[Ctrl] + [P]： プロパティ
[F2]： 名前の変更
[Ctrl] + [N]： 新規フォルダーまたは子メニューを作成
[Ctrl] + [Enter]： フォルダーを開く※2
[Backspace]： 親フォルダーを展開※3
[Ctrl] + [T]： ツールチップON/OFF
[Alt] + [Enter]： シェルのメニューを表示
[Ctrl] + [M]： 全般メニューを表示

※2 「フォルダーを開く」の動作
選択アイテムがフォルダーの場合はそのフォルダーを、フォルダー以外の場合は親フォルダーをエクスプローラーでオープンします。

※3 「親フォルダーを展開」の動作
サブメニューの場合は選択アイテムの親フォルダーを、フォルダー展開メニューの場合はメニュー表示しているフォルダーの親フォルダーをメニュー展開します。
フォルダー展開メニュー専用
[Ctrl] + [X]： 切り取り
[Ctrl] + [C]： コピー
[Ctrl] + [Shift] + [C]： パス名をコピー
[Ctrl] + [V]： 貼り付け
